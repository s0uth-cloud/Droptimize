# Droptimize

Droptimize is a courier management system, with the goal to make it easier for courier company admins, like J&T Express, to track and manage their employees.

This repository is dedicated to the development of the Droptimize mobile application for drivers. The mobile application helps drivers with finding the most efficient route to deliver parcels.
